# GAM – Génies Afrique Médias

## 1. Project Overview

This project, **GAM (Génies Afrique Médias)**, is a digital independent media platform launched by AFRITECK INSTITUT. It aims to be a leading **Web TV and digital media reference in Africa**, highlighting talent, innovation, and positive initiatives.

GAM is designed as a **Progressive Web Application (PWA)** built on a modern Headless architecture, featuring:
-   A robust backend for structured and secure editorial management.
-   A reactive frontend for a fluid user experience, similar to a mobile application.

**Slogan:** *"Révéler les talents, informer l’Afrique"* (Revealing talents, informing Africa)

**Mission:** To promote African innovation, education, culture, and talents through in-depth articles, interviews, and video reports.

## 2. Technical Architecture

### Stack

**Backend (API & Administration)**
-   Django + Wagtail CMS
-   Advanced administration interface for journalists

**Frontend (User Interface)**
-   React.js via Next.js (This project is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app)).
-   Server Side Rendering (SSR) for Google News and SEO optimization.

**Database**
-   PostgreSQL

**Video Hosting**
-   YouTube (Official GAM channel)
